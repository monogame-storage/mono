# THE DARK ROOM - Agent 02 Brief

## BMAD-GDS Brief

**Title:** THE DARK ROOM
**Genre:** Mystery Adventure (Top-Down Visual Map Exploration)
**Platform:** Mono Fantasy Console (160x120, 2-bit grayscale)

### Concept
Wake up in a dark room with no memory. Explore interconnected rooms rendered as top-down visual maps. Move a character around furniture, doors, and items. Find clues, collect inventory items, solve puzzles, and escape.

### Core Mechanics
- **Top-down movement:** D-pad moves character around rooms rendered as visual layouts
- **Interaction:** Z button examines/picks up nearby objects
- **Inventory:** X button opens inventory overlay; select items to use
- **Visibility:** Limited light radius around player; rooms start dark, get brighter as you find light sources
- **Puzzles:** Key-lock, combination codes, item combinations

### Room Layout
1. **Bedroom** - Start here. Bed, nightstand, locked door, dresser
2. **Hallway** - Connects rooms. Painting with clue, locked cabinet
3. **Study** - Desk with journal, safe with combination lock, bookshelf
4. **Kitchen** - Drawer with knife, fridge with note, back door (final exit)

### Inventory Items
- Flashlight (increases visibility)
- Key (unlocks hallway cabinet)
- Journal page (contains safe combination hint)
- Safe code paper (combination for study safe)
- Master key (from safe, opens kitchen back door)

### Win Condition
Escape through the kitchen back door with the master key.

### Demo Mode
Title screen with auto-play demo showing character wandering rooms.

### Sound Design
- Footstep sounds on movement
- Door open/close sounds
- Item pickup chimes
- Ambient tension drone
- Puzzle solve fanfare

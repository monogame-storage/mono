# THE DARK ROOM: PUZZLES

A puzzle-driven escape room. Wake in darkness. Five rooms, five unique puzzle mechanics. Solve each to progress deeper into the mystery. Minimal text -- interaction tells the story.

## Controls
- **D-pad**: Move cursor / select elements
- **A button**: Interact / confirm
- **B button**: Cancel / back
- **Start**: Pause

## Puzzle Rooms
1. **Combination Lock** -- Rotate dials to match a hidden code (clue on the wall)
2. **Sliding Tiles** -- Rearrange a 3x3 grid to form a symbol
3. **Wire Connect** -- Draw paths linking colored endpoints on a grid
4. **Pattern Match** -- Reproduce a flashing light sequence
5. **Sequence Memory** -- Remember and replay an increasingly long tone sequence

## Gameplay
- Each solved puzzle reveals a clue fragment and unlocks the next room
- Atmospheric sound: ambient hum, click feedback, success/fail tones
- Solving all five puzzles reveals the final truth and escape
- Demo mode plays automatically after idle on title screen

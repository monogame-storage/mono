## Agent 11 — THE DARK ROOM: Whispers in the Dark

**Base:** Agent 04 (Horror — top-down stalker, flashlight, battery, jump scares)
**Absorbed:** Agent 01 (Rich narrative text, typewriter effect, journal pages, word-wrap)

### Concept
A horror exploration game where you navigate dark rooms with a flashlight while a stalker entity hunts you. Now enhanced with a rich narrative layer: typewriter-revealed room descriptions, discoverable journal fragments that piece together your backstory as Subject 17 in the LETHE memory-erasure program, and atmospheric text overlays that deepen the dread.

### Merged Features
- **From #04:** Tile-based room exploration, flashlight cone with battery drain/flicker, stalker entity AI with chase/patrol, collision, keys/doors, jump scare system, particles, heartbeat audio, ambient sounds, vignette
- **From #01:** Typewriter text effect for all narrative text, word-wrapped rich room descriptions, journal page collectibles with lore reveals, inventory examine with text overlay, narrative connecting rooms thematically

### Controls
- D-Pad: Move
- A: Interact / advance text
- B: Toggle flashlight
- START: Start game
- SELECT: Pause

### Technical
- `mode(2)`, 2-bit palette (0-3)
- Single file, GAME-STANDARD compliant
- Demo mode with idle timeout
- Sound effects via note/noise

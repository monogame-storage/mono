# DUSKHOLD

## Title
Duskhold

## Short Description
A brutal roguelike dungeon crawler with procedural depths and permadeath.

## Long Description
Duskhold is a turn-based roguelike dungeon crawler where every run is a descent into the unknown. Procedurally generated dungeons ensure that no two expeditions are alike -- each floor reshuffles corridors, chambers, traps, and treasure into unpredictable configurations that demand careful exploration and tactical thinking.

A true field-of-view system keeps you honest. Torchlight only reaches so far, and what lurks beyond the edge of your vision can end a run in an instant. Six distinct enemy types -- from shambling undead to cunning wraiths -- each carry their own movement patterns and attack behaviors, forcing you to adapt your strategy floor by floor.

Scavenge weapons, armor, potions, and scrolls scattered throughout the halls. Every item pickup is a decision: equip now for an immediate edge, or hold it for a deeper floor where the stakes are higher? There are no save points and no second chances. Permadeath means each choice carries real weight, and a single misstep can erase an hour of hard-won progress.

Duskhold rewards patience, punishes greed, and delivers the kind of tense, emergent storytelling that only a true roguelike can offer. Steel yourself, light your torch, and see how deep you can go.

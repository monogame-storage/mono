# ROOK TOWER

## Concept
Tower defense on a chess board. Rooks attack along ranks and files (horizontal/vertical lines). Enemy pawns march along a winding path toward your king. Place rooks to create kill zones using straight-line fire. Upgrade rooks to queens for diagonal coverage.

## Chess Connection
- The grid IS an 8x8 chess board with algebraic notation (a-h, 1-8)
- Rooks fire along their rank and file — true to chess movement
- Enemies are pawns that advance forward
- Queen upgrade adds diagonal attack — true to queen movement
- King sits at the end of the path as your base

## Core Mechanics
- Tap empty squares to place rooks (cost: gold)
- Rooks auto-fire at pawns in their rank/file
- Earn gold from defeated pawns
- Upgrade rooks to queens for diagonal + straight coverage
- Waves of pawns increase in speed and HP
- King has 3 lives — game over when all lost

## Visual Style
- 1-bit black and white only
- Checkerboard grid with chess notation labels
- Rook drawn as crenellated tower shape
- Queen drawn as crowned piece
- Pawns as small rounded pieces
- Laser/bolt lines show attacks along rank/file

## Controls
- D-pad: move cursor on grid
- A button: place rook / upgrade to queen
- START: begin game
- SELECT: pause
- Touch: tap to place/upgrade

## Sound Design
- Note on rook firing
- Noise burst on pawn death
- Low tone when pawn reaches king
- Victory jingle on wave clear

## Demo Mode
- Auto-places towers and plays waves
- 7-segment clock display
- Returns to title on any input

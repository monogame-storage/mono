# Mono Console Spec

## 철학

강력한 제약조건 안에서 창의력을 극대화한다.
제약이 통일된 미학을 만들고, 누구나 게임을 쉽게 만들 수 있게 한다.

## 디스플레이

| 항목 | 스펙 |
|------|------|
| 해상도 | 160×120 고정 |
| 프레임레이트 | 30fps 고정 |

## 컬러

그레이스케일 전용. 부트 시 비트 수를 선택하며, 이후 변경 불가.

| 비트 | 색 수 | 인덱스 범위 | 팔레트 |
|------|--------|-------------|--------|
| 1 (기본) | 2 | 0-1 | 검정, 흰색 |
| 2 | 4 | 0-3 | 0, 85, 170, 255 균등 분배 |
| 4 | 16 | 0-F | 0~255 균등 16단계 |

## VRAM

| 항목 | 스펙 |
|------|------|
| 구조 | `Uint8Array[160×120]` (19,200 bytes) |
| 단위 | 1픽셀 = 1바이트 (컬러 인덱스 저장) |
| 렌더링 | flush 시 팔레트 룩업 → canvas putImageData |

```
Lua script
  ↓ pix(), rect(), circ(), text()...
VRAM (Uint8Array[19200])  ← 본체
  ↓ flush()
Canvas 2D                 ← 디스플레이 (교체 가능)
```

- VRAM은 canvas와 독립. 헤드리스(Node.js) 환경에서도 동작.
- flush 대상은 교체 가능 (putImageData → WebGL texImage2D 등).

## VRAM 접근

| 함수 | 설명 |
|------|------|
| `pix(x,y,c)` | 픽셀 쓰기 |
| `gpix(x,y)` | 픽셀 읽기 (컬러 인덱스 반환) |
| `vrow(y)` | 행 덤프 (hex string, 160자) |
| `vdump()` | 전체 덤프 (hex string, 120행) |

## 그래픽 함수

| 함수 | 설명 |
|------|------|
| `cls(c)` | 화면 지우기 |
| `pix(x,y,c)` | 픽셀 |
| `gpix(x,y)` | 픽셀 읽기 |
| `line(x0,y0,x1,y1,c)` | 선 (Bresenham) |
| `rect(x,y,w,h,c)` | 사각형 테두리 |
| `rectf(x,y,w,h,c)` | 채운 사각형 |
| `circ(x,y,r,c)` | 원 테두리 (midpoint) |
| `circf(x,y,r,c)` | 채운 원 |
| `text(str,x,y,c)` | 비트맵 텍스트 (4×7 폰트) |

## 입력

### 추상 입력 (버튼)

| 함수 | 설명 |
|------|------|
| `btn(key)` | 누르고 있는지 |
| `btnp(key)` | 이번 프레임에 눌렸는지 |

키: `up`, `down`, `left`, `right`, `a`, `b`, `start`

엔진 예약키:
- `Space` — pause 토글 (엔진 레벨)
- `1` — 디버그 모드 토글

키보드 매핑:

| 버튼 | 키 |
|------|-----|
| 방향키 | Arrow / WASD |
| A | Z |
| B | X |
| Start | Enter |

### 마우스/터치 (로우 입력)

| 함수 | 설명 |
|------|------|
| `mouse_x()` | 커서 x (0~159) |
| `mouse_y()` | 커서 y (0~119) |
| `mouse_btn()` | 마우스/터치 누름 여부 |

터치는 마우스로 자동 매핑.

### 중력센서 (로우 입력)

| 함수 | 설명 |
|------|------|
| `accel_x()` | 좌우 기울기 |
| `accel_y()` | 앞뒤 기울기 |
| `accel_z()` | 상하 가속도 |

모바일 기기 전용. 미지원 환경에서는 0 반환.

### 입력 추상화

엔진이 키보드/마우스/터치/중력센서를 `btn()` 레이어로 자동 매핑 가능:
- 터치: 화면 좌/우 영역 탭 → `left`/`right`
- 중력: 기울기 → 방향키

게임은 `btn()`만 써도 모든 입력 장치에서 동작하거나, 로우 입력을 직접 사용할 수 있음.

## 사운드

제한 없음. Web Audio API 기반으로 자유롭게 사용.

## 게임 루프

```
boot → Lua 로드 → _init() 1회 → [_update() + _draw() + flush] @30fps
```

## 스크립트

| 항목 | 스펙 |
|------|------|
| 언어 | Lua 5.4 |
| VM | Wasmoon (로컬 번들) |
| 콜백 | `_init()`, `_update()`, `_draw()` |

## 전역 상수 (Lua)

| 이름 | 값 |
|------|-----|
| `SCREEN_W` | 160 |
| `SCREEN_H` | 120 |
| `COLORS` | 팔레트 크기 (2, 4, 16) |

## 부트 API (JS)

```js
Mono.boot("screen", {
  game: "main.lua",
  colors: 1          // 1, 2, 4 (기본: 1)
})
```

## 비트맵 폰트

- 4×7 픽셀, 모노스페이스
- 대문자 A-Z, 숫자 0-9, 기호 `.,:!?-+/*#()='"<>_`
- 글자 간격: 1px (총 5px/글자)
